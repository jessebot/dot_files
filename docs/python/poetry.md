# Poetry
[poetry](https://python-poetry.org/) is:
> Python packaging and dependency management made easy


## Tips and Tricks

Clearing cache:

```bash
poetry cache list
```

Poetry environment files on macOS default to this directory:
```bash
/Users/$(whoami)/Library/Caches/pypoetry/virtualenvs/
```
